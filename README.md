# ScorpionOS
OS for BitBurner

## How to install ?
1. Copy the files [get-os.js]{https://raw.githubusercontent.com/UpsilonScorpi/ScorpionOS/main/get-os.js} in a script in your game
2. Run this script

## Functionality :
1. UI with :
   1. Control panel
   2. List of server with script runing
   3. Network tree
   4. Guide
2. Contract solving
3. Hacknet node optimization
4. Hacking
